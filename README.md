# Choral extension for Visual Studio Code

Support for the [Choral programming language](https://choral-lang.org) inside of Visual Studio Code. Enjoy!

## Features

- Syntax highlighting.

- Run task.

- Error highlighting 

## Instalation

Simply installing this extension will ensure that a choral installation will be found or downloaded if missing from system. 

If you wish to manually decide which choral installation is used, set `CHORAL_HOME`.
